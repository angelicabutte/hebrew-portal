Order of Operations

Your daily workflow, step by step.

Step 1: Prepare Vocabulary

Make sure your vocab file is formatted correctly and saved in the proper folder.

Step 2: Run Main Script

cd ~/Downloads
python3 hebrew_anki_CLEAN_FINAL.py

Step 3: Choose/Replace Media

Use the preview script to select your preferred image/audio options.

python3 image_preview_no_dupes_v2.py

Step 4: Copy Media to Anki

cp media to Anki collection

Step 5: Verify Deck in Anki

Open Anki, check that media loads correctly on cards, then start studying.

Daily Pre-Flight Checklist

  • Vocab file verified
  • Main script executed
  • Media reviewed and replaced
  • Media copied to Anki folder
  • Deck verified in Anki
